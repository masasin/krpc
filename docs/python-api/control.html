Control¶

class Control¶

Used to manipulate the controls of a vessel. This includes adjusting the throttle, enabling/disabling systems such as SAS and RCS, or altering the direction in which the vessel is pointing.

sas¶

Gets or sets the state of SAS.

Return type:bool
Returns:True if SAS is enabled, False if it is not.
rcs¶

Gets or sets the state of RCS.

Return type:bool
Returns:True if RCS is enabled, False if it is not.
gear¶

Gets or sets the state of the gear/landing legs.

Return type:bool
Returns:True if gear/landing legs are lowered, False if not.
lights¶

Gets or sets the state of the lights.

Return type:bool
Returns:True if lights are enabled, False if they are not.
brakes¶

Gets or sets the state of the wheel brakes.

Return type:bool
Returns:True if wheel brakes are enabled, False if they are not.
abort¶

Gets or sets the state of the abort action group.

Type:bool
throttle¶

Gets or sets to state of the throttle. A value between 0 and 1.

Type:float
pitch¶

Gets or sets the state of the pitch control (equivalent to the w and s keys) [1]. A value between -1 and 1.

Type:float
yaw¶

Gets or sets the state of the yaw control (equivalent to the a and d keys) [1]. A value between -1 and 1.

Type:float
roll¶

Gets or sets the state of the roll control (equivalent to the q and e keys) [1]. A value between -1 and 1.

Type:float
forward¶

Gets or sets the state of the forward translational control (equivalent to the h and n keys) [1]. A value between -1 and 1.

Type:float
up¶

Gets or sets the state of the up translational control (equivalent to the i and k keys) [1]. A value between -1 and 1.

Type:float
sideways¶

Gets or sets the state of the sideways translational control (equivalent to the j and l keys) [1]. A value between -1 and 1.

Type:float
wheel_throttle¶

Gets or sets the state of the wheel throttle [1]. A value between -1 and 1. A value of 1 rotates the wheels fowards, a value of -1 rotates the wheels backwards.

Type:float
wheel_steering¶

Gets or sets the state of the wheel steering [1]. A value between -1 and 1. A value of 1 steers to the left, and a value of -1 steers to the right.

Type:float
current_stage¶

Gets the current stage of the vessel. Corresponds to the stage number in the in-game UI.

Return type:int16
activate_next_stage()¶

Activates the next stage. Equivalent to pressing the space bar in-game.

Return type:list ( Vessel )
Returns:A list of vessel objects that are jettisoned from the active vessel.
get_action_group(group)¶

Returns True if the given action group (a value between 0 and 9 inclusive) is enabled.

Ptype group:uint16
Return type:bool
set_action_group(group, state)¶

Sets the state of the given action group (a value between 0 and 9 inclusive).

Ptype group:uint16
Ptype state:bool
toggle_action_group(group)¶

Toggles the state of the given action group (a value between 0 and 9 inclusive).

Ptype group:uint16
add_node(ut, prograde = 0, normal = 0, radial = 0)¶

Creates a maneuver node at the given universal time, and returns a Node object that can be used to modify it. Optionally sets the magnitude of the delta-v for the maneuver node in the prograde, normal and radial directions.

Parameters:
  • ut (float) – universal time of the maneuver node
  • prograde (float) – delta-v in the prograde direction
  • normal (float) – delta-v in the normal direction
  • radial (float) – delta-v in the radial direction
Return type:

Node
nodes¶

Gets a list of all existing maneuver nodes, ordered by time from first to last.

Return type:list ( Node )
remove_nodes()¶

Removes all maneuver nodes.

Footnotes

[1](1, 2, 3, 4, 5, 6, 7, 8) The control input will persist until the client that requested it disconnects. If multiple clients set a control input, they are added together and clamped to the range [-1,1].
