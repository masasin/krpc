Orbit¶

class Orbit¶

Describes an orbit. For example, the orbit of a vessel, obtained by calling Vessel.orbit(), or a celestial body, obtained by calling CelestialBody.orbit().

body¶

Gets the celestial body (e.g. planet or moon) around which the object is orbiting.

Return type:CelestialBody
apoapsis¶

Gets the apoapsis of the orbit, in meters, from the center of mass of the body being orbited.

Note

For the apoapsis altitude reported on the in-game map view, use Orbit.apoapsis_altitude.

Return type:double
periapsis¶

Gets the periapsis of the orbit, in meters, from the center of mass of the body being orbited.

Return type:double

Note

For the periapsis altitude reported on the in-game map view, use Orbit.periapsis_altitude.

apoapsis_altitude¶

Gets the apoapsis of the orbit, in meters, above sea level of the body being orbited.

Return type:double

Note

This is equal to Orbit.apoapsis minus the equatorial radius of the body.

periapsis_altitude¶

Gets the periapsis of the orbit, in meters, above sea level of the body being orbited.

Return type:double

Note

This is equal to Orbit.periapsis minus the equatorial radius of the body.

semi_major_axis¶

Gets the semi-major axis of the orbit, in meters.

Return type:double
semi_minor_axis¶

Gets the semi-minor axis of the orbit, in meters.

Return type:double
radius¶

Gets the current radius of the orbit, in meters. This is the distance between the center of mass of the object in orbit, and the center of mass of the body around which it is orbiting.

Return type:double

Note

This value will change over time if the orbit is elliptical.

speed¶

Gets the current orbital speed of the object in meters per second.

Return type:double

Note

This value will change over time if the orbit is elliptical.

period¶

Gets the orbital period, in seconds.

Return type:double
time_to_apoapsis¶

Gets the time until the object reaches apoapsis, in seconds.

Return type:double
time_to_periapsis¶

Gets the time until the object reaches periapsis, in seconds.

Return type:double
eccentricity¶

Gets the eccentricity of the orbit.

Return type:double
inclination¶

Gets the inclination of the orbit, in radians.

Return type:double
longitude_of_ascending_node¶

Gets the longitude of the ascending node, in radians.

Return type:double
argument_of_periapsis¶

Gets the argument of periapsis, in radians.

Return type:double
mean_anomaly_at_epoch¶

Gets the mean anomaly at epoch.

Return type:double
epoch¶

Gets the time since the epoch (the point at which the mean anomaly at epoch was measured, in seconds.

Return type:double
mean_anomaly¶

Gets the mean anomaly.

Return type:double
eccentric_anomaly¶

Gets the eccentric anomaly.

Return type:double
reference_plane_normal(reference_frame)¶

Gets the unit direction vector that is normal to the orbits reference plane, in the given reference frame. The reference plane is the plane from which the orbits inclination is measured.

Parameters:reference_frame (ReferenceFrame) –
Return type:Vector3
reference_plane_direction(reference_frame)¶

Gets the unit direction vector from which the orbits longitude of ascending node is measured, in the given reference frame.

Parameters:reference_frame (ReferenceFrame) –
Return type:Vector3
time_to_soi_change¶

Gets the time until the object changes sphere of influence, in seconds. Returns NaN if the object is not going to change sphere of influence.

Return type:double
next_orbit¶

If the object is going to change sphere of influence in the future, returns the new orbit after the change. Otherwise returns None.

Return type:Orbit
