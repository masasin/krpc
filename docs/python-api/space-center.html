SpaceCenter¶

class SpaceCenter¶

The SpaceCenter service provides functionality to interact with Kerbal Space Program. This includes controlling the active vessel, managing it’s resources, planning maneuver nodes and auto-piloting.

active_vessel¶

Gets the currently active vessel.

Return type:Vessel
vessels¶

Gets a list of all the vessels in the game.

Return type:list ( Vessel )
bodies¶

Gets a dictionary of all celestial bodies (planets, moons, etc.) in the game, keyed by the name of the body.

Return type:dict ( string, CelestialBody )
target_body¶

Gets or sets the currently targeted celestial body.

Return type:CelestialBody
target_vessel¶

Gets or sets the currently targeted vessel.

Return type:Vessel
target_docking_port¶

Gets or sets the currently targeted docking port.

Return type:DockingPort
clear_target()¶

Clears the current target.

ut¶

Gets the current universal time in seconds.

Return type:double
g¶

Gets the value of the gravitational constant G in \(N(m/kg)^2\).

Return type:double
warp_to(ut, max_rate = 100000)¶

Uses time acceleration to warp to the specified time. Automatically uses regular or physical time warp as appropriate. For example, physical time warp is used when the active vessel is travelling through an atmosphere. When using physical time warp, the warp rate is at most 2x.

Parameters:
  • ut (float) – The universal time to warp to, in seconds
  • max_rate (float) – The maximum warp rate to use
Returns:

When the time warp is complete.

Converts a position vector from one reference frame to another.

  • from (ReferenceFrame) – The reference frame that the position vector is in.

Returns:

Parameters:

  • to (ReferenceFrame) – The reference frame to covert the rotation to.

Return type:

Quaternion
transform_velocity(position, velocity, from, to)¶

Converts a velocity vector (acting at the specified position vector) from one reference frame to another. The position vector is required to take the relative angular velocity of the reference frames into account.

Parameters:
  • position (Vector3) – Position vector in reference frame from.
  • velocity (Vector3) – Velocity vector in reference frame from.
  • from (ReferenceFrame) – The reference frame that the position and velocity vectors are in.
  • to (ReferenceFrame) – The reference frame to covert the velocity vector to.
Returns:

The corresponding velocity in reference frame to.
Return type:

Vector3
far_available¶

Return type:bool
remote_tech_available¶

Gets whether RemoteTech is installed.

Return type:bool
draw_direction(direction, reference_frame, color[, length])¶

Draw a direction vector on the active vessel.

Parameters:
  • direction (Vector3) – Direction to draw the line in.
  • reference_frame (ReferenceFrame) – Reference frame that the direction is in.
  • color (Vector3) – The color to use for the line, as an r,g,b color.
  • length (float) – The length of the line. Defaults to 10.
clear_directions()¶

Remove all directions currently being drawn.
